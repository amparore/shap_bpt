Minimal examples
================

The main package repository includes compact notebooks demonstrating the core
API and the main hierarchy choices.

Included notebooks
------------------

* ``examples/ImageNET_BPT.ipynb``
* ``examples/ImageNET_BPT_AA.ipynb``
* ``examples/VisualizeBPT.ipynb``

Suggested use in the docs
-------------------------

These notebooks are ideal for:

* quick API orientation,
* learning the difference between BPT and axis-aligned explanations,
* understanding BPT visualization before running larger experiments.
